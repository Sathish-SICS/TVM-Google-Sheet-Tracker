⚙️ One-time Setup (Free)
  1. Go to console.cloud.google.com → Sign in with your Google account
  2. Click Select a project → New Project → give any name → Create
  3. Go to APIs & Services → Library → Search "Google Sheets API" → Enable
  4. Same Library → Search "Google Drive API" → Enable
  5. Go to APIs & Services → Credentials → + Create Credentials → OAuth 2.0 Client ID
  6. Application type: Web application → Under Authorised JavaScript origins add: https://localhost AND the URL where you host this file
  7. Click Create → Copy the Client ID (ends in .apps.googleusercontent.com)
  8. Paste it below and click Save
This tool is 100% free. No charges. Google Cloud APIs used here are in the free tier.

Setup & Help
📦 How your data is stored

All your sheet records are saved in a Google Sheet called "TVM Tracker DB" in your own Google Drive. This means:

✅ Same data on home computer + office computer
✅ Stored in your Google One 100GB account
✅ Never lost — backed up automatically by Google
✅ 100% free, no third-party storage

🔄 Auto-refresh

The dashboard reads all active Google Sheets every 5 minutes automatically. You can also click Refresh Now in the top bar at any time.

The countdown timer in the top bar shows when the next auto-refresh will happen.

📋 Block name matching

When reading a Google Sheet, the dashboard scans every row in every tab looking for block names in Tamil or English. It checks if other cells in that row are filled to determine status:

🟢 நிரப்பினர் (Filled) — 4 or more cells filled
🟡 பகுதி (Partial) — 2–3 cells filled
🔴 நிரப்பவில்லை (Unfilled) — only block name or empty
